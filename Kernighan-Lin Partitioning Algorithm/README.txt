Description of Files in Folder -

main.cpp: Contains all functions for implementing algorithm, writing to console, and loading in data.

case1.txt and case2.txt: Contains example cases of format that is read by program. The first line has
			 two numbers - the number of nodes followed by the number of edges. Each of the
			 following lines has two numbers that are indices of two nodes of an edge.

Output -
	Two partitions, each of which is described by the indices of its member nodes.
	Cut set, represented by the edges in the cut, and edge is denoted by a pair of node indicies.
