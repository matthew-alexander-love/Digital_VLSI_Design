Description of Files in Folder -
	
main.cpp: Contains functions for getting data from file as well high level implementation of Stockmeyers
	  Algorithm.

trees.cpp: Contains functions for creating, navigating, manipulating, and searching tree from polish 
           expression.

trees.h: Contains Node struct definition as well as definition of all functions in trees.cpp.

case.txt: Contains an example case of format that is read by program. 
	  First line is a polish expression, every following line contains the dimensions for a 
          corresponding block.

Output -
	Height and width of the optimized floorplan.
	Orientation of each block in optimized floorplan.
